    Целью изучения дисциплины “Защита информации” является рассмотрение важнейших принципов построения криптосистем, изучение основных алгоритмов шифрования данных и криптографических протоколов.

    В процессе изучения данной дисциплины Вам необходимо:

    • изучить представленный теоретический материал;
    • выполнить три лабораторные работы;
    • выполнить контрольную работу в соответствии с заданием по индивидуальному варианту;
    • сдать экзамен по дисциплине.

    Внимательно прочтите следующие рекомендации:

    1. Сначала Вам необходимо выполнить три лабораторных работы. Задание общее для всех. Результаты каждой лабораторной работы должны содержать файл программы и файл с результатами работы программы.
    2. Лабораторные работы должны быть высланы со страниц нашего сайта. После проверки работы преподавателем Вы получите рецензию с оценкой (зачет/незачет) и возможными замечаниями по представленному решению для исправления или доработки. В этом случае после доработки Вы должны повторно выслать лабораторную работу на проверку.
    3. После того, как Вы получите оценку «зачет» по всем трем лабораторным работам, необходимо начать выполнение контрольной работы.
    4. Номер варианта Вашего индивидуального задания контрольной работы соответствует последней цифре Вашего пароля. Результаты выполненной контрольной работы должны включать текст задания, соответствующий именно Вашему варианту, файл с текстом программы, подробное описание работы программы и результаты работы программы. Отчет с описанием работы программы должен быть в формате текстового редактора Word, размером шрифта 14 пунктов.
    5. Контрольная работа должна быть выслана со страниц нашего сайта. С порядком отправки выполненных работ можно ознакомиться в инструкции по отправке, доступной по ссылке.
    6. После проверки работы преподавателем Вы получите рецензию с оценкой и возможными замечаниями по представленному решению для исправления или доработки. В этом случае после доработки Вы должны повторно выслать контрольную работу на проверку.
    7. Вы будете допущены к экзамену только после получения оценки «зачет» по трем лабораторным работам и зачет за выполнение контрольной работы.
    8. Сдача экзамена осуществляется по билетам, которые выдаются системой дистанционного обучения индивидуально случайным образом. Ответ на билет должен включать в себя текст задания и выбранные варианты ответов. После проверки ответа Вы получите письмо с оценкой.
